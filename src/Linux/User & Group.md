# User & Group

- Group Commands
    
    **List all groups:**
    ```bash
     ~: groups
    ```
    
    **List a group members:**
    ```bash
     ~: getent group g1
    ```

- Users Commands
    
    **List all users:**
    ```bash
    ~: users
    ```
    
    **List all group membership of a user:**
    ```bash
     ~: groups   user1
     ~: id user1
    ```
